# dva-mall
目标：express+mongoDB+antd+dva+react
开发过程：antd+dva+react+mockjs

## 使用

#### 安装依赖
```
npm install
```
#### 启动
```
npm start
```
#### 打包
```
npm run build
```
#### 执行单元测试
```
npm run test
```
#### 生成测试报告
```
npm run coverage
```

### 规则
* 所有的代码必须经过 eslint 的校验
* 所有的页面排版必须一致
* 编辑器的格式化请参看 .editorconfig 配置, 编辑器请自行安装editorconfig插件
